===================
Gestión de Usuarios
===================

La gestión de usuarios  en la Plataforma AUTANA  están estructurados desde tres puntos de vista:

	* Usuarios contribuyentes.
	* Usuarios del Fondo Público.
	* Usuarios solicitantes/beneficiarios de financiamiento.


Usuarios contribuyentes
-------------------------------------------------

Los usuarios de los contribuyentes podrán acceder a las siguientes aplicaciones:


1. Pasos para crear una cuenta contribuyente: 
++++++++++++++++++++++++++++++++++++++++++++++

La aplicación AUTANA crea una cuenta de acceso para una nuevo contribuyente.
	
   #. Acceda a la página de enlace a la aplicación, en el sitio web de FIDETEL.

   #. Seleccione la opción ¿Aún no se ha Registrado? de la página inicial del espacio de contribuyentes. Observe la :ref:`figura1`
	
       .. _figura1:
       .. figure:: _static/imagenes/img1.png
          :align: center
          :height: 600px
          :alt: acceso a Autana

          **Figura 1. Acceso a AUTANA**

   #. El sistema desplegará un formulario como se puede ver en la :ref:`figura2`. Todos los datos son obligatorios, por esta razón es importante 
      no omitir los campos solicitados.

       .. note::
	  Es importante que tome en cuenta:

	 * El RIF debe corresponder con el formato válido para el SENIAT. 
	 * El formato de correo electrónico debe corresponder a dominios válidos. 
	 * El nivel de seguridad de su contraseña debe ser lo más alto posible. (El color rojo le indicará que el nivel de seguridad 
	   es bajo, el color amarillo le indicará que el nivel de seguridad es medio y el color verde le indicará que el nivel de 
	   seguridad es alto). 
	 * El código Captcha ingresado debe ser igual al presentado en la imagen. 
	 * Todos los campos solicitados son obligatorios. 
       .. 
 	
       .. _figura2:
       .. figure:: _static/imagenes/img2.png
          :align: center
          :height: 600px
      
          **Figura 2. Formulario de creación de la cuenta contribuyente**
       
   #. Luego de llenar todos los campos del formulario, presione la opción **REGISTRAR**.
   #. Recibirá en su correo electrónico un enlace que le permitirá la creación de su contraseña, como se muestra en la :ref:`figura3`

       .. _figura3:
       .. figure:: _static/imagenes/img3.png
          :align: center
          :height: 200px 

          **Figura 3. Enlace para creación de contraseña**

   #. Se le mostrará un mensaje indicándole que se ha enviado un correo electrónico para crear su contraseña. Ver :ref:`figura4`.

       .. _figura4:
       .. figure:: _static/imagenes/img4.png
          :align: center
          :height: 200px 

          **Figura 4.  Mensaje para creación de contraseña**

   #. Ingrese en el enlace que se le suministra para crear su contraseña. Llene todos los campos solicitados y luego presione el 
      botón **ACEPTAR** como se observa en la :ref:`figura5`.

       .. _figura5:
       .. figure:: _static/imagenes/img5.png
          :align: center
          :height: 250px 

          **Figura 5.  Creación de contraseña**

   #. Si su registro se ha realizado satisfactoriamente, será redireccionado a la página Espacio de Contribuyentes, como se observa 
      en la :ref:`figura6`. 

       .. _figura6:
       .. figure:: _static/imagenes/img6.png
          :align: center

2. ¿Cómo ingresar  a la aplicación AUTANA (Espacio Contribuyentes)?

Esta operación permite el acceso del Usuario Contribuyente en el sistema, para ello, debe estar previamente registrado. Al ingresar satisfactoriamente en AUTANA se le desplegará su menú de opciones. Adicionalmente, si el usuario debe algún pago, su pantalla de inicio tendrá un listado de los pagos pendientes por realizar.

   #. Presione el enlace Operadoras de Telecomunicaciones, en la dirección `el sitio web de FIDETEL <http://www.fidetel.gob.ve>`_ .

   #. Se le presentará un formulario de ingreso con los siguientes campos: RIF, Contraseña y Validación Captcha.

   #. Llene todos los campos del formulario y luego presione la opción **INGRESAR** como se observa en la :ref:`figura7`	 

       .. _figura7:
       .. figure:: _static/imagenes/img7.png
          :align: center
          :height: 400px 

          **Figura 7. Ingreso al sistema AUTANA**
  
       .. note::

          Es importante que tome en cuenta:

          * El código Captcha ingresado debe ser igual al presentado en la imagen. 
          * Todos los campos solicitados son obligatorios.
       ..
   
   #. Luego de validar los datos ingresados, la aplicación le direccionará a la página Espacio de Contribuyentes. Ver :ref:`figura8`

       .. _figura8:
       .. figure:: _static/imagenes/img8.png
          :align: center

3. ¿Cómo recuperar la contraseña?
++++++++++++++++++++++++++++++++++

Permite al usuario recuperar su contraseña para acceder al sistema.

   #. Seleccione la opción ¿Olvidó su Contraseña? que aparece en el formulario de ingreso, tal como se observa en la :ref:`figura9`

       .. _figura9:
       .. figure:: _static/imagenes/img9.png
          :align: center
          :height: 430px 

          **Figura 9. Formulario de ingreso a AUTANA**

   #. El sistema le solicitará los siguientes datos: RIF, correo electrónico y validación Captcha. 
   #. Luego de llenar todos los campos del formulario, presione el botón **ACEPTAR** como se aprecia en la :ref:`figura10`

       .. _figura10:
       .. figure:: _static/imagenes/img10.png
          :align: center
          :height: 430px 

          **Figura 10. Formulario de ingreso para recuperar contraseña**
   
   #. El sistema le enviará un correo electrónico, con un vínculo que le permitirá colocar una nueva contraseña. :ref:`figura11`
		
       .. _figura11:
       .. figure:: _static/imagenes/img11.png
          :align: center
          :height: 250px 

          **Figura 11. Enlace para ingresar nueva contraseña.**

       .. note:: 
	  Es importante que tome en cuenta:

	  * El nivel de seguridad de su contraseña debe ser lo más alto posible, ya que así evitará que puedan acceder fácilmente a su cuenta.
	  * Incluya en su contraseña caracteres alfabéticos, numéricos y especiales (ejemplo: coma, puntos, etc)  
	    para tener un nivel de seguridad alto.
	  * El nivel de seguridad se mostrará en color rojo cuando sea bajo, color amarillo en nivel medio y color 
	    verde cuando obtenga un nivel alto indicándosele también un porcentaje.
	  * Todos los campos solicitados son obligatorios.
	  * En caso de haber perdido su correo electrónico, contacte al Fondo para actualizarlo.	
       ..

   #. Al acceder al enlace enviado a su correo electrónico la aplicación le mostrará una planilla, tal como se observa en la :ref:`Figura12`, 
      en la  cual debe introducir su nueva contraseña dos veces para confirmarla y luego presionar el botón **ACEPTAR** para realizar el 
      cambio de contraseña.

       .. _figura12:
       .. figure:: _static/imagenes/img12.png
          :align: center
          :height: 430px 

          **Figura 12. Formulario de ingreso de nueva contraseña.**

5. ¿Cómo registrar los datos del contribuyente? 
++++++++++++++++++++++++++++++++++++++++++++++++

Permite hacer el registro de los datos del contribuyente a la que pertenece el usuario. 

   #. Seleccione la opción del Menú Cuenta de Usuario, Contribuyente Ver :ref:`Figura16`	

       .. _figura16:
       .. figure:: _static/imagenes/img16.png
          :align: center
          :height: 250px 

          **Figura 16. Menú contribuyente.**

   #. Se le mostrará un formulario solicitando: Datos Básicos, Domicilio Fiscal, datos del representante legal y de 
      algunas personas de  contacto de la empresa, como se observa en la :ref:`Figura17`

      * *Datos Básicos*: Ingrese la razón social del contribuyente, el correo electrónico, algunos datos del registro mercantil 
	(actividad económica, fecha de registro, número, tomo y folio), la fecha de inicio de operaciones y número telefónico. 
	Para añadir varios números telefónicos, presione el botón **AÑADIR**. Si el contribuyente es el resultado de 
	la fusión con otro contribuyente, marque el recuadro **RESULTADO DE LA FUSIÓN DE OTRAS** del cual se desplegará un campo para que escriba la información acerca de la empresa fusionada.
	
      * *Domicilio Fiscal*: Se le mostrarán listas desplegables para que seleccione el Estado, el Municipio y la Parroquia, 
	además debe ingresar ciudad, dirección y zona postal. 

      * *Representante legal*: Ingrese el nombre, apellido, cédula de identidad, correo electrónico y teléfonos.

      * *Contacto* de la empresa: Ingrese el nombre, apellido, cédula, correo electrónico y número telefónico de algún contacto 
	del contribuyente. Si desea añadir otro contacto, presione la opción **AGREGAR CONTACTO** e ingrese los datos mencionados anteriormente.

       .. _figura17:
       .. figure:: _static/imagenes/img17.png
          :align: center
          :height: 750px 

          **Figura 17. Formulario de datos menú contribuyente.**

   #. Luego de llenar todos los campos del formulario presione la opción **GUARDAR** El sistema validará la información y la 
      registrará exitosamente. Ver :ref:`Figura18`

       .. _figura18:
       .. figure:: _static/imagenes/img18.png
          :align: center
          :height: 520px 

          **Figura 18. Datos del contribuyente.**

       .. note:: Es importante que tome en cuenta:
	  
	  * Todos los campos solicitados son obligatorios,  excepto los detalles de fusión entre empresas. 
	  * Al validar la información se desplegará  el menú del contribuyente (lado izquierdo).

Usuarios del Fondo Público
----------------------------

El personal encargado por parte del Fondo Público podrá acceder a las siguientes aplicaciones:

+++++++++++++++++++++++++++++++++++

Esta aplicación permite al usuario del Fondo acceder a la plataforma AUTANA.

   #. Presione el icono de la aplicación AUTANA que se encuentra en la página web de FIDETEL; se le mostrará un 
      formulario en el cual debe ingresar el usuario y la contraseña como se observa en la :ref:`Figura23`

       .. _figura23:
       .. figure:: _static/imagenes/img23.png
          :align: center
          :height: 490px 

          **Figura 23. Acceso a la plataforma AUTANA.**

   #. Llene todos los campos del formulario y presione el botón **INGRESAR**

   #. El sistema lo ingresará a la página del funcionario del Fondo, como se observa en la :ref:`Figura24`

       .. _figura24:
       .. figure:: _static/imagenes/img24.png
          :align: center
          :height: 430px 

          **Figura 24. Ingreso a la plataforma AUTANA. (usuario del Fondo)**
       
       .. note:: Es importante que tome en cuenta:

          * Todos los campos son obligatorios.
       ..

2. ¿Cómo recuperar la contraseña?
++++++++++++++++++++++++++++++++++

Esta aplicación permite recuperar la contraseña del usuario del Fondo.

   #. Ingrese a la aplicación AUTANA ver :ref:`Figura23`, seleccione la opción ¿Olvidó su contraseña?, como se 
      muestra en la :ref:`Figura25`  

       .. _figura25:
       .. figure:: _static/imagenes/img25.png
          :align: center
          :height: 520px 

          **Figura 25. Selección de opción olvido de contraseña.** 

   #. Ingrese el usuario y correo electrónico al cual se le enviará un vinculo para que acceda al sistema Ver :ref:`Figura26` y 
      presione la opción **ACEPTAR**

       .. _figura26:
       .. figure:: _static/imagenes/img26.png
          :align: center
          :height: 400px 

          **Figura 26. Formulario para modificación de contraseña.**

   #. Se le mostrará un mensaje indicando que se le ha enviado un correo electrónico para recuperar su contraseña.

   #. Ingrese a su correo electrónico y presione el vínculo que le fue enviado. Llene el formulario con las opciones 
      nueva contraseña y confirmar contraseña.

   #. Luego presione el botón **ACEPTAR** y se mostrará en pantalla la página del usuario del Fondo Ver :ref:`Figura24`

       .. note:: Es importante que tome en cuenta:

          * Todos los campos son obligatorios.
	  * Las dos contraseñas deben ser iguales.
