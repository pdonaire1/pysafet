Gestión de Usuarios¶

La gestión de usuarios en la Plataforma AUTANA están estructurados desde tres puntos de vista:

  • Usuarios contribuyentes.
  • Usuarios del Fondo Público.
  • Usuarios solicitantes/beneficiarios de financiamiento.

Usuarios contribuyentes¶

Los usuarios de los contribuyentes podrán acceder a las siguientes aplicaciones:

1. Pasos para crear una cuenta contribuyente:¶

La aplicación AUTANA crea una cuenta de acceso para una nuevo contribuyente.

  1. Acceda a la página de enlace a la aplicación, en el sitio web de FIDETEL.

  2. Seleccione la opción ¿Aún no se ha Registrado? de la página inicial del espacio de contribuyentes. Observe la Figura 1. Acceso a AUTANA

    Figura 1. Acceso a AUTANA

  3. El sistema desplegará un formulario como se puede ver en la Figura 2. Formulario de creación de la cuenta contribuyente. Todos los datos son obligatorios, por esta razón es importante no omitir los campos solicitados.

    Nota

    Es importante que tome en cuenta:

    • El RIF debe corresponder con el formato válido para el SENIAT.
    • El formato de correo electrónico debe corresponder a dominios válidos.
    • El nivel de seguridad de su contraseña debe ser lo más alto posible. (El color rojo le indicará que el nivel de seguridad es bajo, el color amarillo le indicará que el nivel de seguridad es medio y el color verde le indicará que el nivel de seguridad es alto).
    • El código Captcha ingresado debe ser igual al presentado en la imagen.
    • Todos los campos solicitados son obligatorios.

    Figura 2. Formulario de creación de la cuenta contribuyente

  4. Luego de llenar todos los campos del formulario, presione la opción REGISTRAR.

  5. Recibirá en su correo electrónico un enlace que le permitirá la creación de su contraseña, como se muestra en la Figura 3. Enlace para creación de contraseña

    Figura 3. Enlace para creación de contraseña

  6. Se le mostrará un mensaje indicándole que se ha enviado un correo electrónico para crear su contraseña. Ver Figura 4. Mensaje para creación de contraseña.

    Figura 4. Mensaje para creación de contraseña

  7. Ingrese en el enlace que se le suministra para crear su contraseña. Llene todos los campos solicitados y luego presione el botón ACEPTAR como se observa en la Figura 5. Creación de contraseña.

    Figura 5. Creación de contraseña

  8. Si su registro se ha realizado satisfactoriamente, será redireccionado a la página Espacio de Contribuyentes, como se observa en la Figura 6. Finalización de registro.

    Figura 6. Finalización de registro

2. ¿Cómo ingresar a la aplicación AUTANA (Espacio Contribuyentes)?¶

Esta operación permite el acceso del Usuario Contribuyente en el sistema, para ello, debe estar previamente registrado. Al ingresar satisfactoriamente en AUTANA se le desplegará su menú de opciones. Adicionalmente, si el usuario debe algún pago, su pantalla de inicio tendrá un listado de los pagos pendientes por realizar.

  1. Presione el enlace Operadoras de Telecomunicaciones, en la dirección el sitio web de FIDETEL .

  2. Se le presentará un formulario de ingreso con los siguientes campos: RIF, Contraseña y Validación Captcha.

  3. Llene todos los campos del formulario y luego presione la opción INGRESAR como se observa en la Figura 7. Ingreso al sistema AUTANA

    Figura 7. Ingreso al sistema AUTANA

    Nota

    Es importante que tome en cuenta:

    • El código Captcha ingresado debe ser igual al presentado en la imagen.
    • Todos los campos solicitados son obligatorios.

  4. Luego de validar los datos ingresados, la aplicación le direccionará a la página Espacio de Contribuyentes. Ver Figura 8. Página espacio de contribuyentes

    Figura 8. Página espacio de contribuyentes

3. ¿Cómo recuperar la contraseña?¶

Permite al usuario recuperar su contraseña para acceder al sistema.

  1. Seleccione la opción ¿Olvidó su Contraseña? que aparece en el formulario de ingreso, tal como se observa en la Figura 9. Formulario de ingreso a AUTANA

    Figura 9. Formulario de ingreso a AUTANA

  2. El sistema le solicitará los siguientes datos: RIF, correo electrónico y validación Captcha.

  3. Luego de llenar todos los campos del formulario, presione el botón ACEPTAR como se aprecia en la Figura 10. Formulario de ingreso para recuperar contraseña

    Figura 10. Formulario de ingreso para recuperar contraseña

  4. El sistema le enviará un correo electrónico, con un vínculo que le permitirá colocar una nueva contraseña. Figura 11. Enlace para ingresar nueva contraseña.

    Figura 11. Enlace para ingresar nueva contraseña.

    Nota

    Es importante que tome en cuenta:

    • El nivel de seguridad de su contraseña debe ser lo más alto posible, ya que así evitará que puedan acceder fácilmente a su cuenta.
    • Incluya en su contraseña caracteres alfabéticos, numéricos y especiales (ejemplo: coma, puntos, etc) para tener un nivel de seguridad alto.
    • El nivel de seguridad se mostrará en color rojo cuando sea bajo, color amarillo en nivel medio y color verde cuando obtenga un nivel alto indicándosele también un porcentaje.
    • Todos los campos solicitados son obligatorios.
    • En caso de haber perdido su correo electrónico, contacte al Fondo para actualizarlo.

  5. Al acceder al enlace enviado a su correo electrónico la aplicación le mostrará una planilla, tal como se observa en la Figura 12. Formulario de ingreso de nueva contraseña., en la cual debe introducir su nueva contraseña dos veces para confirmarla y luego presionar el botón ACEPTAR para realizar el cambio de contraseña.

    Figura 12. Formulario de ingreso de nueva contraseña.

  6. Si la información suministrada es válida, será direccionado a la página Espacio de Contribuyentes. ver Figura 8. Página espacio de contribuyentes

    Figura 14. Formulario de modificación de contraseña.

  3. Luego de llenar todos los campos, presione la opción GUARDAR

  4. Si la información ha sido registrada se presentará un mensaje satisfactorio, como se observa en Ver Figura 15. Mensaje de modificación de contraseña.

    Figura 15. Mensaje de modificación de contraseña.

    Nota

    Es importante que tome en cuenta:

    • El nivel de seguridad de su contraseña debe ser lo más alto posible, ya que así evitará que puedan acceder fácilmente a su cuenta.
    • Incluya en su contraseña caracteres alfabéticos, numéricos y especiales (ejemplo: coma, puntos, etc) para tener un nivel de seguridad alto.
    • El nivel de seguridad se mostrará en color rojo cuando sea bajo, color amarillo en nivel medio y color verde cuando obtenga un nivel alto indicándole también un porcentaje.

5. ¿Cómo registrar los datos del contribuyente?¶

Permite hacer el registro de los datos del contribuyente a la que pertenece el usuario.

  1. Seleccione la opción del Menú Cuenta de Usuario, Contribuyente Ver Figura 16. Menú contribuyente.

    Figura 16. Menú contribuyente.

  2. Se le mostrará un formulario solicitando: Datos Básicos, Domicilio Fiscal, datos del representante legal y de algunas personas de contacto de la empresa, como se observa en la Figura 17. Formulario de datos menú contribuyente.

    • Datos Básicos: Ingrese la razón social del contribuyente, el correo electrónico, algunos datos del registro mercantil (actividad económica, fecha de registro, número, tomo y folio), la fecha de inicio de operaciones y número telefónico. Para añadir varios números telefónicos, presione el botón AÑADIR. Si el contribuyente es el resultado de la fusión con otro contribuyente, marque el recuadro RESULTADO DE LA FUSIÓN DE OTRAS del cual se desplegará un campo para que escriba la información acerca de la empresa fusionada.
    • Domicilio Fiscal: Se le mostrarán listas desplegables para que seleccione el Estado, el Municipio y la Parroquia, además debe ingresar ciudad, dirección y zona postal.
    • Representante legal: Ingrese el nombre, apellido, cédula de identidad, correo electrónico y teléfonos.
    • Contacto de la empresa: Ingrese el nombre, apellido, cédula, correo electrónico y número telefónico de algún contacto del contribuyente. Si desea añadir otro contacto, presione la opción AGREGAR CONTACTO e ingrese los datos mencionados anteriormente.

    Figura 17. Formulario de datos menú contribuyente.

  3. Luego de llenar todos los campos del formulario presione la opción GUARDAR El sistema validará la información y la registrará exitosamente. Ver Figura 18. Datos del contribuyente.

    Figura 18. Datos del contribuyente.

    Nota

    Es importante que tome en cuenta:

    • Todos los campos solicitados son obligatorios, excepto los detalles de fusión entre empresas.
    • Al validar la información se desplegará el menú del contribuyente (lado izquierdo).

6. ¿Cómo consultar los datos del contribuyente?¶

Permite consultar los datos del contribuyente que han sido registrados.

  1. Seleccione la opción del Menú Cuenta de Usuario, Contribuyente Ver Figura 16. Menú contribuyente.
  2. El sistema presenta por pantalla los datos del Contribuyente Ver Figura 18. Datos del contribuyente.

7. ¿Cómo modificar los datos de la contribuyente?¶

Permite editar los datos del contribuyente registrado.

  1. Seleccione la opción del Menú Cuenta de Usuario, Contribuyente Ver Figura 16. Menú contribuyente.. El sistema muestra los datos del contribuyente registrados.

  2. Seleccione la opción EDITAR

  3. El sistema activará la información registrada para permitir su modificación. Realice los cambios que considere pertinentes y presione la opción GUARDAR Ver Figura 19. Información registrada activada para modificación.

    Figura 19. Información registrada activada para modificación.

    Nota

    Es importante que tome en cuenta:

    • Todos los campos solicitados son obligatorios, excepto los detalles de fusión entre empresas.
    • Si desea incorporar más de una persona de contacto, presione la opción agregar contacto.

Permite crear los datos del usuario contribuyente previamente registrado.

  1. Ingrese en el menú Cuenta de Usuario, Usuario Contribuyente Ver Figura 20. Menú usuario/contribuyente.

    Figura 20. Menú usuario/contribuyente.

  2. Se solicitarán los siguientes datos en el formulario: nombres, apellidos, documento de identidad, correo electrónico, cargo y teléfono. Ver Figura 21. Formulario de datos menú usuario/contribuyente.

    Figura 21. Formulario de datos menú usuario/contribuyente.

  3. Luego de llenar todos los campos del formulario presione la opción GUARDAR . El sistema validará la información y registrará los datos exitosamente. Ver Figura 22. Datos menú usuario/contribuyente.

    Figura 22. Datos menú usuario/contribuyente.

    Nota

    Es importante que tome en cuenta:

    • Todos los campos solicitados son obligatorios.

9. ¿Cómo consultar los datos del usuario contribuyente?¶

Permite visualizar los datos registrados del usuario contribuyente.

  1. Seleccione la opción del Menú cuenta de usuario, Usuario Contribuyente Ver Figura 20. Menú usuario/contribuyente.
  2. El sistema presenta en la pantalla los datos del usuario Contribuyente Ver Figura 22. Datos menú usuario/contribuyente.

10. ¿Cómo modificar los datos del usuario contribuyente?¶

Permite editar los datos de usuario del contribuyente.

  1. En la pantalla Datos del usuario seleccione el icono EDITAR

  2. Modifique los datos que desea actualizar.

  3. Presione la opción GUARDAR

    Nota

    Es importante que tome en cuenta:

    • Todos los campos solicitados son obligatorios.

Usuarios del Fondo Público¶

El personal encargado por parte del Fondo Público podrá acceder a las siguientes aplicaciones:

1. ¿Cómo ingresar a la aplicación?¶

Esta aplicación permite al usuario del Fondo acceder a la plataforma AUTANA.

  1. Presione el icono de la aplicación AUTANA que se encuentra en la página web de FIDETEL; se le mostrará un formulario en el cual debe ingresar el usuario y la contraseña como se observa en la Figura 23. Acceso a la plataforma AUTANA.

    Figura 23. Acceso a la plataforma AUTANA.

  2. Llene todos los campos del formulario y presione el botón INGRESAR

  3. El sistema lo ingresará a la página del funcionario del Fondo, como se observa en la Figura 24. Ingreso a la plataforma AUTANA. (usuario del Fondo)

    Figura 24. Ingreso a la plataforma AUTANA. (usuario del Fondo)

    Nota

    Es importante que tome en cuenta:

    • Todos los campos son obligatorios.

2. ¿Cómo recuperar la contraseña?¶

Esta aplicación permite recuperar la contraseña del usuario del Fondo.

  1. Ingrese a la aplicación AUTANA ver Figura 23. Acceso a la plataforma AUTANA., seleccione la opción ¿Olvidó su contraseña?, como se muestra en la Figura 25. Selección de opción olvido de contraseña.

    Figura 25. Selección de opción olvido de contraseña.

  2. Ingrese el usuario y correo electrónico al cual se le enviará un vinculo para que acceda al sistema Ver Figura 26. Formulario para modificación de contraseña. y presione la opción ACEPTAR

    Figura 26. Formulario para modificación de contraseña.

  3. Se le mostrará un mensaje indicando que se le ha enviado un correo electrónico para recuperar su contraseña.

  4. Ingrese a su correo electrónico y presione el vínculo que le fue enviado. Llene el formulario con las opciones nueva contraseña y confirmar contraseña.

  5. Luego presione el botón ACEPTAR y se mostrará en pantalla la página del usuario del Fondo Ver Figura 24. Ingreso a la plataforma AUTANA. (usuario del Fondo)

    Nota

    Es importante que tome en cuenta:

    • Todos los campos son obligatorios.
    • Las dos contraseñas deben ser iguales.

3. ¿Cómo registrar los datos del usuario del Fondo?¶

Permite crear satisfactoriamente los datos del usuario del Fondo.

  1. Presione la opción del menú Usuario del Fondo como se observa en la Figura 27. Menú usuario del Fondo.

    Figura 27. Menú usuario del Fondo.

  2. Le aparecerá un formulario en el cual debe ingresar los siguientes datos: nombre, apellido, cédula, correo electrónico, fecha de nacimiento, sexo y estado civil como se observa en la Figura 28. Formulario de registro de datos del usuario del Fondo.

    Figura 28. Formulario de registro de datos del usuario del Fondo.

  3. Luego de ingresar todos los datos en el formulario, presione la opción GUARDAR . Los datos serán registrados exitosamente.

    Nota

    Es importante que tome en cuenta:

    • Todos los campos son obligatorios.

4. ¿Cómo consultar los datos del usuario del Fondo?¶

Esta aplicación le permite consultar los datos del usuario que han sido registrados en el sistema.

  1. Presione la opción del menú Usuario del Fondo Ver Figura 27. Menú usuario del Fondo.

  2. Se le mostrará en la pantalla los datos que fueron registrados como usuario del Fondo, como se observa en la Figura 29. Datos del usuario del Fondo.

    Figura 29. Datos del usuario del Fondo.

5. ¿Cómo modificar los datos del usuario del Fondo?¶

Permite modificar los datos del usuario del Fondo que han sido previamente registrados.

  1. Presione la opción del menú Usuario del Fondo Ver Figura 27. Menú usuario del Fondo.
  2. Se le mostrará en la pantalla los datos que fueron registrados como usuario del Fondo Ver Figura 29. Datos del usuario del Fondo.
  3. Luego presione la opción EDITAR
  4. Modifique los datos que desee y presione la opción GUARDAR

6. ¿Cómo modificar la contraseña del usuario del Fondo?¶

El sistema permite modificar la contraseña de acceso del usuario.

  1. Presione la opción del menú Contraseña, como se observa en la Figura 30. Menú contraseña.

    Figura 30. Menú contraseña.

  2. Se le mostrará un formulario en el cual deberá ingresar su contraseña actual, su nueva contraseña y realizar la confirmación de la misma Ver Figura 31. Formulario para modificación de contraseña.

    Figura 31. Formulario para modificación de contraseña.

  3. Ingrese todos los datos en el formulario y presione la opción GUARDAR

  4. Se le mostrará un mensaje indicándole que la contraseña se ha modificado exitosamente.

    Figura 43. Formulario de modificación de datos del usuario (Autana – Proyectos)

  4. Luego de llenar el formulario, presione la opción REGISTRAR

  5. El sistema validará los datos y si éstos son correctos creará el usuario solicitante, mostrándole un mensaje de registro exitoso en la página de bienvenida a Autana – Proyectos Figura 45. Mensaje exitoso de registro del usuario solicitante..

  4. El sistema le enviará un correo electrónico, con un vínculo que le permitirá colocar una nueva contraseña.

Presione el icono de la aplicación AUTANA que se encuentra en la página web de FIDETEL; PROPONENTES Y BENEFICIARIOS DE PROYECTOS se le mostrará un formulario en la página de bienvenida de Autana – Proyectos, tal como se observa en la Figura 32. Acceso a Autana – Proyectos (usuario del Fondo)..

  3. Seleccione del menú ubicado al lado izquierdo la opción usuario Tal como se observa en Figura 48. Menú usuario del proyecto..
